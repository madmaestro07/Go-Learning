# App for Black jack

## Functions of the APP 

| Function | Description |
|----------|-------------|
|newDeck | Create & retrun a list of playing cards |

| shuffle | Shuffles all the cards in decl |
| deal | create a "hand" of cards. |
| saveToFile | save a list of cards in file on local |
| newDeckFromFile | load a list of cards from a local file on machine |
